# Abdul Raheman Shaikh — Portfolio

A premium, dark-themed portfolio website showcasing projects, skills, and services. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Live Demo

[View Live](https://your-vercel-url.vercel.app)

## Features

- **Sticky Card Projects Section** — Cards stack on scroll with smooth scale-up animation and 3D tilt on hover
- **Dark/Light Theme Toggle** — Persistent theme switching with smooth transitions
- **Mobile Navigation** — Hamburger menu with slide-in drawer and overlay
- **Animated Particle Background** — Floating particles with mouse interaction
- **Typing Effect** — Typewriter animation in hero section
- **Scroll Reveal** — Elements animate into view via IntersectionObserver
- **3D Card Tilt** — Perspective transform on mouse hover for project cards
- **Custom Cursor** — Gold ring + dot cursor that expands on interactive elements
- **Back to Top** — Smooth scroll-to-top button
- **Download CV** — Resume download with click tracking
- **Fully Responsive** — Mobile, tablet, and desktop layouts

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, flexbox, grid, animations, media queries
- **JavaScript** — Vanilla ES6, DOM manipulation, IntersectionObserver, requestAnimationFrame
- **Fonts** — Kanit (headings), Satoshi (body) via Google Fonts
